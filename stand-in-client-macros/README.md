# stand-in-client-macros

Procedural macros for **[stand-in-client](https://crates.io/crates/stand-in-client)**.

This crate provides the `#[mcp_client]` attribute macro, which generates typed
MCP client stubs from a trait definition. See the
[stand-in-client](https://crates.io/crates/stand-in-client) crate for usage
and the complete SDK.
